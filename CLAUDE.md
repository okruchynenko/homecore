# HomeCore — Project Guidelines

## Git commits
Every commit must include a clear description of what was changed and why.

Commit message format:
```
<short title (imperative, max 72 chars)>

- What was added / changed / fixed (bullet points)
- Why the change was made (if not obvious)

Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>
```

Examples of good titles:
- `Add pagination to product catalog`
- `Replace tag pills with multi-select listbox`
- `Fix wishlist badge not updating after modal toggle`
